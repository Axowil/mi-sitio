Recursión

Matemáticas Discretas | Tiempo de lectura: 12 minutos

La recursión es una técnica fundamental en matemáticas y ciencias de la computación donde una función se define en términos de sí misma. Es una herramienta poderosa para resolver problemas que pueden descomponerse en subproblemas más simples de la misma naturaleza.

¿Qué es la Recursión?

Una definición recursiva consta de dos partes esenciales:

Componentes de una Definición Recursiva

  • Caso Base: La condición que detiene la recursión y proporciona un valor directo
  • Caso Recursivo: La regla que reduce el problema hacia el caso base

Ejemplo Clásico: Factorial

El factorial de un número natural n (denotado n!) es el producto de todos los enteros positivos menores o iguales a n.

Definición Matemática

factorial(n) = { 1, si n = 0 (caso base) n × factorial(n-1), si n > 0 (caso recursivo) }

Implementación en Código

function factorial(n) { // Caso base if (n === 0 || n === 1) { return 1; } // Caso recursivo return n * factorial(n - 1); }

Ejemplo: Calculando 5!

factorial(5) = 5 × factorial(4)

= 5 × 4 × factorial(3)

= 5 × 4 × 3 × factorial(2)

= 5 × 4 × 3 × 2 × factorial(1)

= 5 × 4 × 3 × 2 × 1

= 120

Secuencia de Fibonacci

La secuencia de Fibonacci es otro ejemplo clásico donde cada número es la suma de los dos anteriores.

Definición Recursiva

fibonacci(n) = { 0, si n = 0 1, si n = 1 fibonacci(n-1) + fibonacci(n-2), si n > 1 }

Implementación

function fibonacci(n) { // Casos base if (n === 0) return 0; if (n === 1) return 1; // Caso recursivo return fibonacci(n - 1) + fibonacci(n - 2); }

Recursión en Estructuras de Datos

Suma de Elementos en un Array

function sumaArray(arr, n) { // Caso base: array vacío if (n <= 0) { return 0; } // Caso recursivo: último elemento + suma del resto return sumaArray(arr, n - 1) + arr[n - 1]; }

Torres de Hanói

Las Torres de Hanói es un problema clásico que se resuelve elegantemente con recursión. El objetivo es mover n discos de una torre a otra usando una torre auxiliar.

function hanoi(n, origen, destino, auxiliar) { if (n === 1) { console.log(`Mover disco de ${origen} a ${destino}`); return; } // Mover n-1 discos a torre auxiliar hanoi(n - 1, origen, auxiliar, destino); // Mover el disco más grande console.log(`Mover disco de ${origen} a ${destino}`); // Mover n-1 discos de auxiliar a destino hanoi(n - 1, auxiliar, destino, origen); }

Recursión vs Iteración

Ventajas de la Recursión

  • Código más limpio y legible para problemas naturalmente recursivos
  • Más fácil de entender para problemas como árboles y grafos
  • Reduce la cantidad de código necesario

⚠️ Desventajas de la Recursión

  • Mayor uso de memoria debido a la pila de llamadas
  • Puede ser más lenta que las soluciones iterativas
  • Riesgo de desbordamiento de pila con recursión profunda
  • Algunas implementaciones recursivas tienen complejidad exponencial

Tipos de Recursión

1. Recursión Directa

Una función se llama directamente a sí misma.

2. Recursión Indirecta

Una función A llama a una función B, que a su vez llama a A.

function par(n) { if (n === 0) return true; return impar(n - 1); } function impar(n) { if (n === 0) return false; return par(n - 1); }

3. Recursión de Cola (Tail Recursion)

La llamada recursiva es la última operación de la función. Algunos lenguajes optimizan este tipo de recursión.

function factorialTail(n, acumulador = 1) { if (n === 0) return acumulador; return factorialTail(n - 1, n * acumulador); }

Pensamiento Recursivo

Para resolver problemas recursivamente, pregúntate:

  1. ¿Cuál es el caso más simple que puedo resolver directamente? (Caso base)
  2. ¿Cómo puedo reducir el problema a una versión más pequeña de sí mismo? (Caso recursivo)
  3. ¿Eventualmente llegaré al caso base?

Aplicaciones de la Recursión

  • Recorrido de árboles y grafos
  • Algoritmos de ordenamiento (QuickSort, MergeSort)
  • Algoritmos de búsqueda (Búsqueda binaria)
  • Programación dinámica
  • Divide y conquista
  • Procesamiento de estructuras anidadas

La recursión es una técnica fundamental que, aunque puede parecer compleja al principio, proporciona soluciones elegantes a muchos problemas en ciencias de la computación y matemáticas.
